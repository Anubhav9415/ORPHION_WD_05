## Technologies Used

- HTML5
- CSS3 (with animations)
- JavaScript (ES6+)
- OpenWeatherMap API
- Font Awesome Icons

## Features Breakdown

### Animations
- Fade-in effects for weather data
- Floating weather icon
- Slide-in animations for city name
- Scale-in effect for temperature
- Hover effects on detail cards
- Loading spinner

### Weather Data Displayed
- Current temperature
- Weather description
- Feels like temperature
- Humidity percentage
- Wind speed
- Atmospheric pressure
- Visibility distance
- Cloudiness percentage
- Sunrise time
- Sunset time

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
